# Vape Store - 电子烟专卖店

一个位于西班牙的电子烟在线销售网站，支持西班牙语和中文双语，采用黑白简约设计风格。

## 功能特性

- **双语支持**: 西班牙语和中文一键切换
- **产品展示**: 展示四种电子烟产品（可扩展）
- **直接支付**: 集成Bizum移动支付，款项直接进入商家账户
- **产品管理**: 管理员可以添加/编辑产品信息
- **响应式设计**: 适配桌面和移动设备
- **简约设计**: 黑白配色，苹果官网风格

4. **冰王 (Elfbar IceKing)** - €16.99

## 支付方式

- **Bizum支付**: 客户直接转账到商家手机号
- **支付流程**:
  1. 客户选择产品并点击"购买"
  2. 系统显示商家的Bizum电话号码和订单号
  3. 客户通过银行App使用Bizum转账
  4. 客户截图支付凭证并发送给商家确认

## 使用方法

### 本地运行
1. 克隆项目
2. 在浏览器中直接打开 `index.html`
3. 或使用本地服务器：
   ```bash
   # Python
   python -m http.server 8000
   
   # Node.js
   npx http-server
   ```

### 自定义配置
1. 在 `script.js` 中修改 `CONFIG.bizumPhone` 为你的手机号
2. 在 `TRANSLATIONS` 对象中更新文本内容
3. 在 `products` 数组中添加/修改产品信息

### 添加新产品
1. 在网站底部"产品管理"区域填写产品信息
2. 点击"添加产品"按钮
3. 新产品将立即显示在产品列表中
